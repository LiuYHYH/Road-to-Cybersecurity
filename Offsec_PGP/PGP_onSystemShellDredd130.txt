cpulimit -l 100 -f -- /bin/sh -p # run this to get shell with root privilege
-l 100: Sets the CPU limit to 100% (basically telling it not to actually limit anything).
-f: Tells cpulimit to "fork" and run the following program.
--: This is the "End of Options" marker. It tells cpulimit that everything after this is the command it should execute, not more flags for cpulimit itself.
/bin/sh -p: This launches the shell. The -p is critical because it tells the shell to preserve the effective user ID (which is root because of the SUID bit).
